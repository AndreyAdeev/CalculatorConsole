# CalculatorConsole
Console calculator

Консольный калькулятор (версия 3.0)
- введите в консоли (путь - папка с файлом Calculator.class) команду: java Calculator;
- нажмите Enter;
- введите первое число;
- нажмите Enter;
- введите оператор сложения, вичитания, умножения или деления (+, -, *, /) или "=" для получения результата;
- нажмите Enter;
- введите число;
- нажмите Enter;
- введите оператор сложения, вичитания, умножения или деления (+, -, *, /) - для повторной операции, 
или введите "=" для получения результата;
- нажмите Enter;

Результат операции между введенными числами отобразится в поле консоли.
