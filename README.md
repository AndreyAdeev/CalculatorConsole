# CalculatorConsole
Console calculator

Консольный калькулятор (версия 1.0)
- введите в консоли (путь - папка с файлом Calculator.class) команду: java Calculator;
- нажмите Enter;
- введите первое число;
- нажмите Enter;
- введите второе число;
- нажмите Enter;
- введите оператор сложения, вичитания, умножения или деления (+, -, *, /);
- нажмите Enter;
Результат операции между двумя введенными числами отобразится в поле консоли.
