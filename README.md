# CalculatorConsole
Console calculator

Консольный калькулятор (версия 2.0)
- введите в консоли (путь - папка с файлом Calculator.class) команду: java Calculator;
- нажмите Enter;
- введите первое число;
- нажмите Enter;
- введите оператор сложения, вичитания, умножения или деления (+, -, *, /);
- нажмите Enter;
- введите число или введите "=" для получения результата;
- нажмите Enter;
- введите оператор сложения, вичитания, умножения или деления (+, -, *, /) - для повторной операции, 
или введите "=" для получения результата;
- нажмите Enter;
- введите число (если до этого вы выбрали оператор), или введите "=" для получения результата;
- нажмите Enter;

Результат операции между введенными числами отобразится в поле консоли.
